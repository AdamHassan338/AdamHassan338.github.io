Bootcamp 2020: space

My first programming project, concluding the 2 week Programming Bootcamp at City (2020). This Processing project demonstrates a comprehensive understanding of the use of arrays, loops, objects and classes. I created a space invader style game where the targets and lasers are both objects storedin arrays so there can be multiple them on screen at a time,then each of them are controled using for each loops. You can either control the spaceship with your mouse or keybord. If you let 3 targets get to the bottom of the screen then its game over.
